üìñ Essential Git Commands

üé§ Narration (7:00-8:30): "Let me show you the essential Git commands I use daily. These include cloning repositories, creating branches, staging and committing changes, pushing to remote, and merging code. Understanding these commands is crucial for any developer working with version control."

Clone & Setup

git clone https://github.com/SagsMan/FUDMA-BOT.git cd FUDMA-BOT git config user.name "Your Name" git config user.email "your@email.com"

Create & Switch Branch

git checkout -b feature/branch-name git branch # List branches git branch -D branch-name # Delete branch

Stage & Commit

git add . # Stage all git add file.txt # Stage specific git commit -m "Your message" git log --oneline # View history

Push & Pull

git push origin branch-name git push -u origin main # Track remote git pull origin main # Fetch & merge git fetch origin # Fetch only

Merge & Rebase

git merge feature-branch git rebase origin/main git merge --squash feature # Squash commits

Undo Changes

git restore file.txt # Discard changes git reset HEAD file.txt # Unstage git revert HEAD # Undo last commit git clean -fd # Remove untracked
